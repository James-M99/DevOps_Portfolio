DevOps Projects

update.sh

This is a bash script that automates the deployment of the website. When executed, it will:

  • Pull the latest code from the GitHub Repository
  • Back up the current site
  • Deploy the new site content
  • Log each step
  • Ensure that permissions are correct
  • Make site scripts be globally accessable

healthcheck.sh

This is a bash script that shows a system and website health check. When executed, it will:

  • Log each action with a timestamp
  • Verify that the web server is running
  • Check if key files exist
  • Moniter disk usage
  • Confirm internet connectivity
  • Report deployed website version

status.sh

This is a bash script that is a website status report designed to give a quick overview of the health and status of the web server. When executed, it will print a report to the terminal with the following:

  • Currently deployed version of the website
  • Disk space used by the website
  • Most recent backup made
  • The last few entries from the site update log
  • The last few entries from the site backup log
  • Server Uptime

WireGuard VPN

I have added a full tunnel VPN to the website through . After downloading the WireGuard app (for mobile) or software (for computer) a user can configure their WireGuard so that the website's IP address is used instead of their personal one.

Server Configuration
Configuration file: /etc/wireguard/wg0.conf 

    
        [Interface]
        PrivateKey = server-private-key
        Address = 10.0.0.1/24
        ListenPort = 51820

        [Peer]
        PublicKey = 
        AllowedIPs = 10.0.0.2/32
Client Configuration (Example) 
    
        [Interface]
        PrivateKey = client-private-key
        Address = 10.0.0.2/32
        DNS = 1.1.1.1


        [Peer]
        PublicKey = server-public-key
        Endpoint = 52.64.27.54:51820
        AllowedIPs = 0.0.0.0/0, ::/0
        PersistentKeepalive = 25
